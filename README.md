## How to build it?

*  首先,你确定是否要自己编译??,如果是,那就接着往下看,不是??后面自己找编译好的....
* 然后你需要的是 GHC 的编译器,草民用的是 7.10.1的 当然还有cabal 
* 然后cabal install 就好了
* 然而并没有确定采用什么开源协议目测将会采用BSD3 :)
* 然后，cabal install 神马。？？ 答案是直接克隆就好了。。
* 目前并没有计划推至 hackage.haskell.org 什么的，所以不要尝试从hackage直接安装。。
* 特殊版本的特殊说明看后面啦。。
